1. 각 set마다 immoral(5000) + sexual(2500) + abuse(2500) = 10000개의 문장 선정
   - 5개의 set (50000) + validation set (5000) = 전체 55000개의 문장 생성

2. 분리된 형태소를 참고하여, 해당 문장의 핵심이 될만한 형태소를 옆 컬럼에 적어서 사전의 base를 구축

3. 만약 의미 단위가 너무 잘게 분리되어 있었다면, 분리된 형태소 + 원본 단어 모두를 적기

4. 애매한 문장은 과감하게 제거하기
   - 각 주제에 맞지 않는 문장
   - 중복된 단어 표현이 너무 많은 문장
   - 성희롱 + 폭언과 같이 다중 라벨이 가능한 문장
